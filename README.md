# pooColours.js

# what?

pooColours is a JavaScript script to randomly generate hex poo colours. The colours are based on:

* `#4E3318`
* `#33220E`
* `#794D1D`
* `#895735`
* `#281100`
* `#553818`
* `#BB9611`

## usage

`pooColour()` will generate a random colour that looks like poo for whenever you need it.
